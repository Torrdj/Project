14/03/2016 - Quantum Quest - Alpha Release

Bonjour !

L'équipe Script Makers est fière de vous annoncer qu'une première version de Quantum Quest est finalement disponible au grand public, malgré le fait qu'elle ne contienne pas grand chose.

Ainsi, dans cette version, vous pourrez lancer le jeu (génial !), il est possible d'incarner un personnage, et de se déplacer dans une carte temporaire.

  • Features :

    • - Personnage (un seul)
    - Menu start
    - Carte temporaire
    - Déplacement
