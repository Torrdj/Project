01/05/2016 - Quantum Quest - Beta Release

Bonjour !

L'équipe Script Makers est fière de vous annoncer qu'une première version de Quantum Quest est finalement disponible au grand public, dans une nouvelle version du jeu !

Dans cette nouvelle version vient se joindre au jeu, les deux nouvelles classes, le serveur et le computer, nous avons retiré les maps temporaires, et ajouté une nouvelle partie de la carte. Se joint à cela une nouveau menu, toujours plus joli ! Il est également possible de se battre en PvP, et PvE (il n'y a pas encore beaucoup d'ennemis, mais ça arrive !)

  • Features :

    • - Nouveau menu
    - Nouvelle interface
    - Nouvelle map
    - Deux nouveaux personnages
    - PvP + PvE
    - Quelques sons
